## 🚀 Instalación

_Clona el repositorio en tu maquina local_

```
git clone git@github.com:enadamgu/vue3-password-generator.git
```

⌨️ Instala las dependencias

```
npm install
```

## ⚙️ Dev Server

```
npm run dev
```

## 📦 Build

```
npm run build
```
